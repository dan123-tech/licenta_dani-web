Place your SIGNED release (or debug) APK here with this exact name:

  fleetshare.apk

Android Studio: open the "android" folder -> Build -> Generate Signed App Bundle / APK -> APK
(or Build -> Build Bundle(s) / APK(s) -> APK for a debug APK for testing).

Copy the built .apk into:  public/downloads/fleetshare.apk
Then commit and redeploy (Vercel, etc.). Without this file, the download link returns 404 and phones cannot install anything.

Website URL: https://YOUR-SITE/downloads/fleetshare.apk

Optional: host the APK elsewhere and set NEXT_PUBLIC_ANDROID_DOWNLOAD_URL in .env to that full HTTPS URL.

Install on phone: open the downloaded file from Notifications or Downloads; allow "Install unknown apps" for Chrome/your browser if Android asks.
